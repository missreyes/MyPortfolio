:: Monthly Update Procedure - BarkerNext JPOC Server

1. Shutdown the JPOC server application by pressing the EXIT button on the MasterApp application window. On the Barker Manager application, select File and then Exit. Click OK on Com Server Warning window.

2. Open Windows Explorer and find the c:\progeny\data directory. In that directory is the bconfig.ini file. Right-Click on the bconfig.ini file and select delete.
3. In Windows Explore note the drive letter assignment of the existing firewire drive It is usually f: or g:. This drive will contain the \LoopingContent directory.
4. Right Click on the \LoopingContent directory and select delete.
5. Plug in the new firewire drive for the monthly update. A new drive letter will appear in the Windows Explorer window.
6. Click on the new drive letter.
7. Right Click on the \chainpromos directory and select copy. Right click on the existing firewire drive letter (f: or g:) and select paste.
8. Right Click on the \customlayoutelements directory and select copy. Right click on the existing firewire drive letter (f: or g:) and select paste.
9. Right Click on the \custompromos directory and select copy. Right click on the existing firewire drive letter (f: or g:) and select paste.
10. Right Click on the \globalpromos directory and select copy. Right click on the existing firewire drive letter (f: or g:) and select paste.
11. Right Click on the \layout directory and select copy. Right click on the existing firewire drive letter (f: or g:) and select paste.
12. Right Click on the \layoutelements directory and select copy. Right click on the existing firewire drive letter (f: or g:) and select paste.
13. Right Click on the \layoutservices directory and select copy. Right click on the existing firewire drive letter (f: or g:) and select paste.
14. Right Click on the \playlist directory and select copy. Right click on the existing firewire drive letter (f: or g:) and select paste.
15. Right Click on the \servicepromos directory and select copy. Right click on the existing firewire drive letter (f: or g:) and select paste.
16. Right Click on the \update directory and select copy. Right click on the existing firewire drive letter (f: or g:) and select paste.
17. Right Click on the \loopingcontent directory and select copy. Right click on the existing firewire drive letter (f: or g:) and select paste (This process will several minutes).
18. If the IMN content is also changing for this months update, proceed to the IMN update section of this document. If not go to step 19.
19. Follow existing procedure for disconnection of firewire drives and remove the monthly update drive.
20. Close windows explorer and doubleclick on the desktop shortcut for MasterApp.
21. When prompted, double-click on the correct hotel handle from the list provided and select OK.
22. Verify system operation. Update Complete

Updating IMN

1. Browse to the new update drive (F: or G:), then double click on the BBXUpdate.exe program in the root directory.
2. The program displays the following dialog:

Where :
Source (Update) drive: the drive designation for the new update drive.
System Destination Drive: the drive designation of where the music executables reside.
MP3 Destination Directory: the directory where music content reside.


The program detects these locations automatically.

3. Verify that the drive selections and directory locations are correct, then click Update.

The program will update all the new content, and executables, then delete the old content.

4. When the update is done, click Exit to exit the program.

5. Follow existing procedure for disconnection of firewire drives and remove the monthly update drive.

6. Close windows explorer and doubleclick on the desktop shortcut for MasterApp.

7. When prompted, double-click on the correct hotel handle from the list provided and select OK.

8. Verify system operation. Update Complete
