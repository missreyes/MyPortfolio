If your looking for a specific Barker ID or Barker Name, you can search for that Barker using the Find Search method. Please follow the directions below:

Directions:
1. Press and hold down the Ctrl key, while holding down the Ctrl key, press and hold the F key

2. You will get a screen that pops up and looks like this:

This is the Find screen for the window, type in the "Find what:" box the Barker Name, or Barker ID

3. Click on the Find Next button. This will find the Barker ID or Barker Name that you entered in the text box.

4. When you click on an image that you would like to see it will open in this window, to go back to this window press on the "BackSpace" Key on your keyboard.
